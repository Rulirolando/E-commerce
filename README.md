# 🛒 E-Commerce Website (Next.js)

cara menjalankan: 
```npm run dev```

Website e-commerce sederhana menggunakan **Next.js (App Router)** dengan fitur:
- Produk
- Kategori
- Login
- Love / Wishlist Produk
- Jual
- Logout
- Daftar
- Keranjang

---

## 🚀 Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
